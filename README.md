# Electronic Dean's Office
## Опис
REST API на базі FastAPI + SQLite. Практична робота з дисципліни
"Безпека інформаційних систем".

## Виконані роботи
### Практична робота №4: Аутентифікація та безпека
- **Хешування паролів**: Впроваджено Bcrypt через бібліотеку `passlib` для безпечного зберігання паролів.
- **Реєстрація**: Ендпоінт `POST /auth/register` з валідацією Email та складним паролем.
- **Вхід (Login)**: Ендпоінт `POST /auth/login` з захистом від атак перебору (Username Enumeration).
- **Валідація**: Використання Pydantic v2 для перевірки вхідних даних.
